# X509设备参数看板 项目文档

## 项目概述

本项目为**充电板生产线设备参数看板系统**，用于实时展示生产线各工位设备数据。前端采用 Vue 3 + Vite 构建，后端基于 .NET 8 + ASP.NET Core Web API，数据库使用 PostgreSQL。

**服务器地址**: `10.1.17.212`

---

### 1.4 通信机制

前端采用**内网优先、降级公网**的请求策略：

```javascript
const fetchWithFallback = async (queryParams) => {
  try {
    // 先尝试内网
    const response = await fetch(apiConfig.internal, {...});
    if (response.ok) return response;
    throw new Error(`内网 HTTP ${response.status}`);
  } catch (error) {
    console.log('内网失败，切换公网...');
    // 降级到公网
    const response = await fetch(apiConfig.external, {...});
    return response;
  }
}
```

## 四、数据库设计

### 4.1 数据库类型

| 项目       | 信息                                                                             |
| ---------- | -------------------------------------------------------------------------------- |
| 数据库类型 | PostgreSQL                                                                       |
| 版本       | 16                                                                               |
| 连接库     | Npgsql.EntityFrameworkCore.PostgreSQL 8.0.0                                      |
| 连接字符串 | `Host=localhost;Port=5432;Database=X509Data;Username=postgres;Password=123456` |

### 4.2 核心表结构

#### 表1: `X509mes_station_list` (站点列表)

| 字段名      | 数据类型     | 描述         | 约束        |
| ----------- | ------------ | ------------ | ----------- |
| StationId   | UUID         | 站点唯一标识 | PRIMARY KEY |
| StationName | VARCHAR(500) | 站点名称     | NOT NULL    |
| StationCode | VARCHAR(100) | 站点编码     | NOT NULL    |
| CreateTime  | TIMESTAMP    | 创建时间     |             |
| UpdateTime  | TIMESTAMP    | 更新时间     |             |

#### 表2: `X509Data` (数据记录)

| 字段名        | 数据类型     | 描述              | 约束                                          |
| ------------- | ------------ | ----------------- | --------------------------------------------- |
| DataId        | UUID         | 数据记录唯一标识  | PRIMARY KEY                                   |
| SNnumber      | VARCHAR(200) | 序列号/产品唯一码 |                                               |
| StationListId | UUID         | 关联站点ID        | FOREIGN KEY → X509mes_station_list.StationId |
| Data          | TEXT         | 测试数据          |                                               |
| RewokTime     | TIMESTAMP    | 返工时间          | NULLABLE                                      |
| CreateTime    | TIMESTAMP    | 创建时间          |                                               |

### 4.3 ER关系图

```
┌─────────────────────────┐       ┌─────────────────────────┐
│   X509mes_station_list  │       │       X509Data           │
├─────────────────────────┤       ├─────────────────────────┤
│ StationId (PK)          │──┐     │ DataId (PK)             │
│ StationName             │  │     │ SNnumber                │
│ StationCode             │  │     │ StationListId (FK) ─────┘
│ CreateTime             │  │     │ Data                    │
│ UpdateTime             │  └───→ │ RewokTime               │
└─────────────────────────┘        │ CreateTime              │
                                   └─────────────────────────┘
```

**关系说明**:

- `X509Data.StationListId` → `X509mes_station_list.StationId`
- 关系类型：**一对多 (1:N)**
- 一个站点可以对应多条数据记录

### 4.4 索引说明

| 表名     | 索引字段      | 类型     | 用途               |
| -------- | ------------- | -------- | ------------------ |
| X509Data | StationListId | 普通索引 | 按站点查询数据记录 |
| X509Data | CreateTime    | 普通索引 | 按时间排序         |
